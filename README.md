# FunnyPieces
Create pieces from image and sort them by position.

Install
-------
```
0. Install PIL on your PC.
1. $git clone https://github.com/manhdaovan/MovingPieces.git
2. $cd /path/to/MovingPieces
3. $pip install -r requirements.txt
```

Run & Fun
---------
```
$python cmdi.py
```
